# BlinkIT-Analysis-Power-BI

An **interactive Power BI dashboard** designed to analyze **sales performance, customer satisfaction, and inventory distribution** for the Blinkit app.  
The dashboard helps identify **key business insights and optimization opportunities** using multiple KPIs and dynamic visualizations.

> **Note:** This dashboard is inspired by and built following this YouTube tutorial by  
> **Chandoo**: https://youtu.be/moZ2cL3a90U

## Dashboard Overview

The dashboard is divided into the following sections:

### 1. Filter Panel
Allows users to dynamically filter the dashboard by:
- Outlet Location Type  
- Outlet Size  
- Item Type  

Multiple selections can be applied to enable more granular analysis.

### 2. KPI Indicators
Displays high-level performance metrics, including:
- Total Sales  
- Average Sales  
- Number of Items  
- Average Customer Rating
  
These KPIs provide a quick of overall business performance.

### 3. Granular Analysis
Based on the selected KPIs (in this case Total Sales), this provides detailed insights through charts showing:
- Total Sales by Fat Content
- Total Sales of Fat Content distribution across Outlet Types
- Total Sales performance by Item Types

### 4. Sales & Outlet Analysis
This section includes:
- **Line Chart:** Total Sales by Outlet Establishment Year  
- **Donut Chart:** Total Sales by Outlet Size  
- **Funnel Chart:** Total Sales by Outlet Location  
- **Table View:** Outlet Type comparison of Total Sales, Number of Items, Average Sales, Average Rating, and Item Visibility.
